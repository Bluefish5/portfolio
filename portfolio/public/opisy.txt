Jest to projekt gry Tetrisa napisanego na Arduino.

Jest to projekt prostego słownika w konsoli napisny w jezyku C z wykorzystaniem listy powiązanej.

Jest to projekt prostej bazy danych pozwalający na dodawania i usuwanie elementów za p	omocą nowoczesnego interfaceu.

Jest to porgram uczący pojedyńczy neuron tak aby nauczył się oddzielać dwa zbiory punktów za pomocą funkcji liniowej. 
        Kody zostały napisane z wykorzystaniem następujących technologii:
        1. Python 3.9.2
        2. numpy
        3. PyQt5
        4. matplotlib

        Narzedzia wykorzystane:
        1. visual studio code
        2. Qt designer 5.11.1
